# ExpenseTracker
Built for privacy with Jetpack Compose, The ExpenseTracker is a secure, privacy-focused mobile application designed for individuals to manage their personal finances effortlessly. ts core differentiator is that it operates 100% offline.ensuring that sensitive financial data never leaves the user's device. To safeguard this data, the app integrates Biometric Authentication
